# 강한 매니저 (Strong Manager)

## 📋 프로젝트 개요

**강한 매니저**는 AI 기반 온라인 교육 관리 시스템으로, 부트캠프 및 화상 교육 환경에서 수강생을 체계적으로 관리하고 교육 운영의 효율성을 극대화하는 대시보드 플랫폼입니다.

### 🎯 기획 의도

- 부트캠프 및 온라인 화상 교육 콘텐츠의 증가에 따른 체계적인 관리 시스템 필요
- 교육운영업체의 인건비 절감 및 사업 확장 지원
- 온라인 교육의 몰입도, 참여율, 집중력 향상
- 수강생 현황 파악 및 업무 자동화를 통한 운영 효율성 증대

## ✨ 주요 기능

### 1. 대시보드 UI/UX
- **3가지 테마 모드**
  - 라이트 모드: 밝은 배경의 기본 테마
  - 다크 모드: 어두운 배경의 야간 작업용 테마
  - 교관 모드: 다크 + 레드 + 화이트 테마 (강렬한 감시 모드)
- **실시간 테마 전환**: 버튼 클릭으로 즉시 테마 변경 가능
- **LocalStorage 활용**: 사용자가 선택한 테마 자동 저장

### 2. 감독 제어 시스템
- **감독 시작/종료**: 실시간 모니터링 활성화/비활성화
- **일시 정지**: 일시적으로 감시 중단
- **긴급 공지**: 전체 수강생에게 즉시 알림 발송

### 3. 실시간 통계
- 총 수강생 수
- 출석 인원
- 결석 인원
- 경고 받은 학생 수

### 4. AI 에이전트 상태 모니터링 및 설정
- **운영 Agent**: 교육 Q&A, 공지사항, 일정 관리
- **학습 Agent**: 수업 질의응답, 내용 요약 (RAG 모델)
- **감시 Agent**: 카메라 화면 감시, 경고 발송 (YOLO 기반)
- **출결 Agent**: 출결 체크 및 정정 요청 자동화
- **⚙️ Agent 세부 설정 기능**
  - 각 Agent별 설정 버튼 (⚙️) 클릭으로 상세 설정 가능
  - Agent 활성화/비활성화
  - API 엔드포인트 설정
  - 언어 설정 (한국어/영어/일본어)
  - Agent별 특수 설정 (응답 속도, 민감도, 알림 빈도 등)
  - 설정값 LocalStorage 자동 저장

### 5. 수강생 실시간 현황
- 8명의 수강생 목록 (엄태홍, 권순우, 정찬희, 이재균, 문소라, 이동현, 권동훈, Sammy)
- 각 수강생의 연락처 정보 관리
- 실시간 상태 표시 (출석/자리비움/오프라인)
- 카메라 상태 모니터링 (켜짐/꺼짐)
- 마지막 확인 시간
- 경고 횟수
- **개별 작업 기능**
  - 📞 **전화 발신**: 수강생에게 직접 전화 연결
  - 💬 **메시지 발송**: 개별 메시지 전송
  - 🔔 **알림 발송**: Teams/SMS 알림 전송

### 6. 출결 현황 관리
- **📊 엑셀 다운로드**: 출결 현황을 Excel 파일로 다운로드
  - 수강생별 출석 상태, 카메라 상태, 경고 횟수 포함
  - 통계 정보 자동 생성 (총 수강생, 출석, 결석, 경고 대상)
  - 출력 일시 및 시스템 정보 포함
- SheetJS (xlsx) 라이브러리 활용

### 7. 월별 출석부
- **📅 월별 출석 현황 표시**
  - 수강생별 일일 출석 상태 시각화
  - 출석/결석/외출/조퇴/지각 구분 표시
  - 주말/공휴일 자동 구분
  - 월 단위 출석 통계 자동 계산
- **📊 출석 통계 대시보드**
  - 평균 출석률 (출석일/총 수업일)
  - 평균 출석시간률 (실제 출석시간/총 수업시간)
  - 총 수업일수 표시
- **📆 월 선택 기능**
  - 최근 6개월 출석부 조회 가능
  - 드롭다운으로 간편한 월 전환
- **💾 월별 출석부 엑셀 다운로드**
  - 선택한 월의 전체 출석 데이터 Export
  - 수강생별 일일 출석 현황 포함
  - 개인별 출석/결석/외출/조퇴/지각 통계
  - 출석률 및 출석시간률 자동 계산
  - 전체 평균 통계 포함

### 8. 활동 로그
- 실시간 시스템 활동 기록
- 시간별 이벤트 로그
- 타입별 분류 (정보/성공/경고/오류)
- 전화, 메시지, 알림 발송 이력 추적

### 9. Teams 수업 일정 조회 시스템
- **📋 예약된 수업 일정 표시**
  - Teams에서 Power Automate로 예약된 수업 자동 동기화
  - Microsoft Teams SDK를 통한 실시간 일정 불러오기
  - **로그 형식**으로 간단하게 한 줄씩 표시
  - 예시: `2025년 1월 2일 목요일 09:00 ~ 18:00, "AI 알고리즘 실습" 수업 모임 예약되었습니다.`
- **🔄 새로고침 기능**
  - 새로고침 버튼으로 최신 일정 동기화
  - 새로운 수업 예약 시 활동 로그 자동 생성
  - 로그 메시지: `'수업명(HH:mm ~ HH:mm)' 모임이 성공적으로 예약되었습니다`
- **🎯 시각적 구분**
  - 오늘 수업: 초록색 배경 + 좌측 초록 보더 + 굵은 글씨
  - 내일 수업: 좌측 파란 보더
  - 예정 수업: 기본 스타일
  - 지난 수업: 반투명 처리 + 회색 아이콘
- **📱 깔끔한 로그 디자인**
  - 아이콘 + 텍스트 형식
  - 스크롤 가능한 컨테이너 (최대 높이 400px)
  - 호버 시 우측 이동 애니메이션

## 🛠 기술 스택

### Frontend
- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인, CSS Variables를 활용한 테마 시스템

### External Libraries
- **Font Awesome 6.4.0**: 아이콘 라이브러리
- **Google Fonts (Inter)**: 타이포그래피
- **SheetJS (xlsx 0.18.5)**: 엑셀 파일 생성 및 다운로드

- **Langchain**: AI Agent 구현
- **OpenAI API**: 자연어 처리 및 RAG 모델
- **YOLO**: 얼굴 인식 및 출결 감시
- **Azure**: VM, Vector Storage, OpenAI API
- **Microsoft Teams SDK**: 팀즈 연동

## 🎨 사용 방법

### 1. 테마 변경
- 우측 상단의 **테마 전환** 버튼 클릭
- 라이트 모드 → 다크 모드 → 교관 모드 순환

### 2. 감독 시작
1. **감독 시작** 버튼 클릭
2. 실시간 모니터링 활성화
3. AI 에이전트 자동 작동
4. 수강생 상태 실시간 업데이트

### 3. 감독 종료
- **감독 종료** 버튼 클릭으로 모니터링 중단

### 4. AI Agent 설정
각 Agent 카드의 우측 상단 **⚙️ 설정 버튼** 클릭:
1. Agent 설정 모달 창이 열립니다
2. 원하는 설정값을 수정합니다:
   - **Agent 활성화**: 토글 스위치로 ON/OFF
   - **API 엔드포인트**: 연동할 API 주소 입력
   - **언어 설정**: 한국어/영어/일본어 선택
   - **Agent별 특수 설정**: 각 Agent의 고유 기능 조정
3. **저장** 버튼 클릭으로 설정 저장
4. 설정은 LocalStorage에 자동 저장되어 다음 접속 시에도 유지됩니다

#### Agent별 주요 설정 항목
- **운영 Agent**: 응답 속도, 자동 응답, 알림 빈도 (분), 최대 대기열 크기
- **학습 Agent**: 응답 속도, RAG 모델, 요약 주기 (분), 컨텍스트 윈도우 크기
- **감시 Agent**: 감지 민감도 (낮음/보통/높음), 얼굴 인식, 자동 경고, 경고 임계값, 체크 주기 (초)
- **출결 Agent**: 자동 승인, 변경 알림, 동기화 주기 (분), 지각 기준 (분)

### 5. 수강생 관리 작업
수강생 테이블의 작업란에서 다양한 작업 수행:
- **📞 전화 버튼**: 수강생에게 직접 전화 발신
- **💬 메시지 버튼**: 개별 메시지 작성 및 발송
- **🔔 알림 버튼**: 실시간 알림 전송

### 6. 출결 현황 다운로드
1. 수강생 현황 섹션 우측 상단의 **출결현황 다운로드** 버튼 클릭
2. Excel 파일 자동 생성 및 다운로드
3. 파일명 형식: `출결현황_YYYY-MM-DD_HH-MM-SS.xlsx`
4. 포함 내용:
   - 수강생별 상세 정보 (번호, 이름, 연락처, 상태, 카메라, 경고)
   - 출결 통계 (총 수강생, 출석, 결석, 자리비움, 경고 대상)
   - 출력 일시 및 시스템 정보

### 7. 월별 출석부 조회 및 다운로드
1. **월 선택**: 월별 출석부 섹션의 드롭다운에서 조회할 월 선택
   - 최근 6개월 출석 데이터 선택 가능
2. **출석 현황 확인**:
   - 수강생별 일일 출석 상태 확인
   - 색상 코드로 출석 상태 구분:
     - 🟢 출석: 초록색
     - 🔴 결석: 빨간색
     - 🟡 외출: 주황색
     - 🟣 조퇴: 보라색
     - 🔴 지각: 분홍색
     - ⚪ 주말: 회색
3. **통계 확인**:
   - 평균 출석률, 평균 출석시간률
   - 각 수강생별 출석/결석/외출/조퇴/지각 횟수
   - 개인별 출석률 및 출석시간률
4. **Excel 다운로드**: 
   - **월별 출석부 다운로드** 버튼 클릭
   - 파일명 형식: `월별출석부_YYYY년MM월_YYYY-MM-DD.xlsx`
   - 포함 내용:
     - 일별 출석 현황 (모든 날짜)
     - 수강생별 출석 통계
     - 전체 평균 통계
     - 출력 일시 및 시스템 정보

### 9. Teams 수업 일정 조회
1. **자동 일정 동기화**:
   - Teams에서 Power Automate로 수업 예약
   - 강한 매니저가 Microsoft Teams SDK로 자동 동기화
   - 예약된 수업 일정 섹션에서 로그 형식으로 확인
2. **새로고침**:
   - 우측 상단 **새로고침** 버튼 클릭
   - Teams에서 새로 예약된 수업 불러오기
   - 새 예약 감지 시 활동 로그에 자동 기록:
     - `'수업명(HH:mm ~ HH:mm)' 모임이 성공적으로 예약되었습니다`
3. **수업 일정 확인**:
   - 로그 형식으로 간단하게 표시
   - 예시: `2025년 1월 2일 목요일 09:00 ~ 18:00, "AI 알고리즘 실습" 수업 모임 예약되었습니다.`
   - 날짜 구분:
     - 🟢 오늘: 초록색 배경 + 굵은 글씨
     - 🔵 내일: 파란색 보더
     - ⚪ 지난 수업: 반투명 처리

### 9. Teams 수업 예약
1. **수업 예약하기**:
   - Teams 수업 예약 섹션에서 입력 폼 작성
   - **수업명**: 강의 제목 입력 (예: Python 기초 강의)
   - **날짜**: 수업 날짜 선택
   - **시작 시간**: 수업 시작 시간 선택
   - **종료 시간**: 수업 종료 시간 선택
   - **Teams 미팅 예약** 버튼 클릭
2. **예약 결과**:
   - 성공 메시지 팝업 표시
   - 활동 로그에 `'수업명(HH:mm ~ HH:mm)' 모임이 성공적으로 예약되었습니다` 기록
   - 예약된 수업 일정에 자동 추가
3. **예약된 수업 관리**:
   - 예약된 수업 일정 카드에서 확인
   - **참여 버튼**: Teams 미팅 즉시 참여 (실제 환경에서 Teams 앱 실행)
   - **삭제 버튼**: 예약 취소
   - 날짜 뱃지로 수업 일정 구분:
     - 🟢 오늘: 당일 수업
     - 🔵 내일: 다음날 수업
     - 🟡 예정: 1주일 이내 수업
     - ⚪ 지난 수업: 종료된 수업
